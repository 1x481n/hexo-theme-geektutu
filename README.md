# Hexo-theme-geektutu

> [极客兔兔的博客](https://geektutu.com)目前使用该主题。 

## 特性

- [ ] 支持标签、归档
- [x] 支持显示目录
- [ ] 增加对gitment或gitalk等基于Github Issue第三方评论的支持。 

## Hexo 基础命令

一些基础命令:

```bash
hexo new post "<post name>" # 创建一篇新文章
hexo clean && hexo generate # 生成静态文件
hexo server # 本地调测
hexo deploy # 自动部署到你配置的仓库的gh-pages分支
```
